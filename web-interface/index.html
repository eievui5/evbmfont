Bitmap Font converter

This is a frontend for evbmfont. You can use it to create .fnt files.

Upload an image below and hit convert. The .fnt file will be returned as text. Copy and paste the generated text into a file and make sure the file extension is .fnt.

Remember that .fnt files require a local copy of the source image. By default, the generated file assumes its source image is adjacent to it in the filesystem. You can change this by editing the 3rd line of the .fnt file.
